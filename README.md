## 💡 프로젝트 소개
- K-MOOC 공식 사이트의 메인 페이지를 클론 코딩한 프로젝트예요!
- 반응형 레이아웃과 인터랙티브 UI를 직접 구현하며 프론트엔드 역량을 키우고자 했어요 😊
- 시맨틱 마크업, 웹 접근성, GSAP 애니메이션 등 실무에서 쓰이는 기술들을 적용해 봤습니다.
- 포트폴리오 목적으로 제작되었으며, Mobile / Tablet / Desktop 환경 모두에서 동작해요.

## 🛠 사용 기술
- **HTML5** - Semantic Markup
- **CSS3** - Flexbox, Grid, CSS Variables
- **Vanilla JavaScript** - ES6+
- **GSAP** - 3.12.2 (Draggable 플러그인 포함)
- **Google Fonts** - Noto Sans KR

## 📁 포함된 항목

```text
k-Mooc-main/
├── index.html           # 메인 HTML 구조
├── css/
│   └── styles.css       # 메인 스타일시트
├── js/
│   ├── script.js        # 메인 UI 인터랙션 로직
│   └── popup-script.js  # 팝업 알림 및 LocalStorage 제어 로직
└── images/              # 프로젝트 에셋 (아이콘, 배너, 썸네일 등)
```

## ✅ 주요 기능
알차게 준비한 기능들이에요 🎉

- **반응형 디자인**: Mobile, Tablet, Desktop 환경에 최적화된 레이아웃
- **웹 접근성 & 시맨틱 마크업**: 스킵 네비게이션, 시맨틱 태그 활용, SEO 최적화
- **팝업 알림 시스템**: LocalStorage 연동으로 24시간 동안 재노출 방지
- **카테고리 드래그**: GSAP Draggable을 적용한 마우스·터치 드래그 탐색
- **콘텐츠 슬라이더**: 마이크로러닝 및 숏폼 섹션의 이전/다음 슬라이드 애니메이션
- **상단 이동 버튼**: 스크롤 위치에 따른 'To Top' 버튼 활성화 및 부드러운 스크롤

## ⚙️ 설치 방법
별도의 의존성 설치 없이 바로 실행할 수 있어요!

로컬 환경에서 실행하는 방법을 단계별로 안내해 드릴게요 😄

1. 저장소를 클론하거나 ZIP으로 다운로드합니다.
   ```bash
   git clone https://github.com/your-username/k-Mooc.git
   ```
2. 프로젝트 폴더로 이동합니다.
3. `index.html` 파일을 최신 웹 브라우저에서 실행합니다.

## 🚀 사용법
이렇게 사용하시면 돼요!

- **기본 실행**: `index.html`을 브라우저에서 바로 열기
- **권장 실행**: VS Code의 **Live Server** 확장 프로그램을 사용하면 LocalStorage 및 비디오 로딩이 원활하게 동작해요

```
VS Code → 확장 프로그램 → Live Server 설치 → index.html에서 우클릭 → "Open with Live Server"
```
